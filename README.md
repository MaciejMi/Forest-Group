## Run Locally

To run Sorter locally, run this command on your git bash:

Linux and macOS:
```bash
sudo git clone https://github.com/MaciejMi/Forest-Group.git
```

Windows:
```bash
git clone https://github.com/MaciejMi/Forest-Group.git
```
